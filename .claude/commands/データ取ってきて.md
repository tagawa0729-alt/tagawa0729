## STEP 1: 対象投稿の特定

`knowledge/post-history.md` を読み込み、以下の条件を全て満たす投稿を抽出する:
1. `metrics_fetched: false`
2. `posted_at` から **24時間以上経過している**（数字が安定するまで待つ）

対象が0件の場合は「取得対象なし」と報告して終了する。

---

## STEP 2: Threads API でエンゲージメントを取得

各対象投稿について以下を実行:

### エンゲージメント取得
```
GET https://graph.threads.net/v1.0/{post_id}?fields=id,text,timestamp,like_count,replies_count,repost_count&access_token={ACCESS_TOKEN}
```

### コメント取得
```
GET https://graph.threads.net/v1.0/{post_id}/replies?fields=text,timestamp&access_token={ACCESS_TOKEN}
```

取得したコメントが **質問かどうか** を判定する:
- 「？」「教えて」「どうしたら」「どれ」が含まれていたら → 質問フラグ: true

### エラー時
- APIエラーの場合は `metrics_fetched: false` のままにして次の投稿へ
- レート制限の場合は残りをスキップして完了報告のみ
