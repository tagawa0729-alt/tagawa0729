## 投稿生成ルール

### 共通
- X投稿: 140字以内（日本語）
- Threads投稿: 200字以内
- 各アカウントにつき3案（角度違い）を生成すること
- 必ず1つのCTA（いいね・保存・フォロー・コメント）を入れる
- アフィリエイトリンク推薦は毎3投稿に1回以下に抑える

### バズりやすい投稿の型（優先度順）
1. **問題提起型**: 「〇〇してる人、実は損してる理由」
2. **リスト型**: 「知らないと損する〇〇選」（奇数推奨）
3. **逆説型**: 「〇〇をやめたら、むしろ〇〇が改善した」
4. **体験談型**: 「〇〇日間やってみた結果」
5. **トレンド乗っかり型**: 今日のトレンドワードを自然に組み込む

### 出力フォーマット（必ずこのJSON形式）
```json
{
  "date": "YYYY-MM-DD",
  "generated_at": "HH:MM",
  "posts": [
    {
      "account_id": 1,
      "account_name": "メンズスキンケア入門",
      "platform": "X",
      "pattern": "問題提起型",
      "post": "投稿本文",
      "trend_ref": "参照したトレンド（なければnull）",
      "cta": "使ったCTA",
      "affiliate_hint": "おすすめアフィリエイト商材（あれば）"
    }
  ]
}
```
